\# 📧 Spam Email Detection using Machine Learning



\## 📌 Project Overview

This project classifies emails/messages as \*\*Spam\*\* or \*\*Ham (Not Spam)\*\* using Machine Learning techniques.



\## 🚀 Features

\- Text preprocessing (cleaning \& formatting)

\- TF-IDF vectorization

\- Naive Bayes classification

\- Spam prediction system



\## 🛠️ Technologies Used

\- Python

\- Pandas

\- Scikit-learn

\- Jupyter Notebook



\## 📂 Project Structure
SpamDetection/

│── data/

│ └── spam.csv

│── notebooks/

│ └── spam\_detection.ipynb

│── README.md


\## ⚙️ How to Run

1\. Install dependencies:
pip install pandas scikit-learn
2. Run Jupyter Notebook:
jupyter notebook

3. Open `spam\_detection.ipynb` and run all cells



\## 📊 Model Used

\- Multinomial Naive Bayes



\## 🎯 Output

\- Predicts whether a message is Spam or Ham



\## 📌 Example

Input: "Win a free ticket now"  

Output: Spam  



\## 📖 Learning Outcomes

\- Understood text preprocessing

\- Learned feature extraction using TF-IDF

\- Built a classification model
